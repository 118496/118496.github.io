Processor en geheugen

Processor

Algemeen

De processor is een onmisbaar onderdeel in een computer. De processor bepaalt namelijk voor een heel groot deel de prestaties van je computer. En daarbij heb je hem ook echt nodig, want zonder processor zal een computer niet werken (niet eens opstarten). Processor afgekort is CPU (Central Processing Unit).

Het hart van de computer

De processor wordt ook wel het hart van de computer genoemd, want de processor is namelijk verantwoordelijk voor alle berekeningen en aansturingen die in een computer plaatsvinden. Oftewel, hoe sneller de processor, hoe meer berekeningen en aansturingen er per seconde kunnen worden verwerkt.

Aantal processorkernen; Vroeger was een processor gewoon één processor. Tegenwoordig kunnen er meerdere processorkernen in één processor zitten. En bij 6 processorkernen noemen we dit een Hexa Core processor.
